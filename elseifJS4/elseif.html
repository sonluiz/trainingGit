Lệnh if else if else trong js

Khi nào sử dụng else if : khi mà có nhiều hơn 1 điều kiện

    Cú pháp lệnh if else if else

    if (điều kiện 1) {
        // điều kiện 1 == true thì code trong {} sẽ chạy
    } else if (điều kiện 2) {
        // điều kiện 2 == true thì code trong {} sẽ chạy
    } else if (điều kiện 3) {
        // điều kiện 3 == true thì code trong {} sẽ chạy
    } else if (điều kiện 4) {
        // điều kiện 4 == true thì code trong {} sẽ chạy
    } else {
        // các trường hợp còn lại nếu tất cả các điều kiện bên trên đều sai thì code {} sẽ chạy
    }
