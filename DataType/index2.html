từ khóa typeof sẽ cho phép kiểm tra kiểu dữ liệu của biến hoặc giá trị

    toán tử so sánh : == ( a == b )
    == so sánh giá trị ( so sánh tương đối )
    === so sánh giá trị và so sánh kiểu dữ liệu ( so sánh tuyệt đối )
    toán tử >=
    toán tử <=
    toán tử ++ ( tăng thêm 1 giá trị )
    toán tử -- ( giảm đi 1 đơn vị )
